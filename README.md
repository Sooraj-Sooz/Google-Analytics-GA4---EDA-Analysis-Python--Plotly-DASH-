# 📊 Google Analytics 4 (GA4) Analysis Project

## 📌 Project Overview

This project focuses on Exploratory Data Analysis (EDA) and marketing performance analysis using Google Analytics 4 (GA4) data for the datascience.one education platform.

The analysis was conducted using Python, Plotly, Dash, and Power Query to evaluate user acquisition, engagement, and conversion performance. The primary objective was to identify opportunities to improve lead quality, enrollment efficiency, and overall marketing ROI.

---

# 🎯 Project Objective

The main objective of this project was to analyze website user behavior and marketing performance to support data-driven growth strategies for the education platform.

The analysis focused on:
- User acquisition performance
- Engagement behavior
- Funnel conversion analysis
- Lead quality assessment
- Enrollment optimization
- Marketing ROI improvement
